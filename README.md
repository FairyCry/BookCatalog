Книжный каталог

Простое веб-приложение для учёта книг. Добавляй, удаляй, ищи и листай по страницам.

Что умеет

- Добавление книги (название, автор, год, жанр)
- Список всех книг с пагинацией (5 книг на страницу)
- Поиск по названию, автору и жанру
- Сортировка "сначала новые" (по Id)
- Удаление книги (кнопка рядом с каждой)
- Фильтр по жанру (выпадающий список)

Как запустить

1. Установи .NET 8 (если ещё нет)
2. Склонируй репозиторий или скачай zip
3. Открой терминал в папке с проектом
4. Введи `dotnet run`
5. Открой браузер и перейди на `http://localhost:...`

Технологии

- C# и ASP.NET Core (Minimal API)
- Entity Framework Core + SQLite
- HTML, CSS, JavaScript (fetch, DOM)

Пара моментов

- База данных создаётся автоматически при первом запуске (файл bookslist.db)
- Если меняешь модель, удали старый .db файл, чтобы EF Core создал таблицы заново
- Навигация: index.html (главная), addBook.HTML (добавление), bookList.HTML (список)
